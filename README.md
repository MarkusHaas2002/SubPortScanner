# SubPortScanner
Scans Subdomains and scans for a specific Port.
## Linux How-To:
Requirements: Python 2 installed.
Run "sudo install.sh"
python SubPortScanner.py [domain] [port]

## Windows How-To:
Requirements: Python 2 in Path installed.
Run win_install.bat
python SubPortScanner.py [domain] [port]
